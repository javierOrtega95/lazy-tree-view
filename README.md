## Lazy Tree View

React component for managing an lazy tree view.

### Features

- Asynchronous children loading.
- Drag-and-drop support for reordering nodes.
- Customizable via props.

### Tech Stack

- **React**
- **Vite**
- **pnpm**

### Getting Started

```bash
cd lazy-tree-view
```

```bash
pnpm install
```

```bash
pnpm dev
```
